Welcome 🎉

This Pathfinding Visualizer helps you understand how graph algorithms work step by step.

How to Use 🛠

  • 👆 Left-click = Set Start, End, or Walls
  • 🖱 Right-click = Cycle terrains (Normal / Grass / Water)
  • ▶ Choose an algorithm and press Run
  • 🧹 Use Clear Grid to reset

Algorithms 🤖

  • BFS – Unweighted shortest path
  • Dijkstra – Weighted shortest path
  • A* – Optimized heuristic search
  • DFS – Depth-first traversal
  • Greedy Best-First – Heuristic only
  • Bidirectional BFS – Double-ended BFS

Features ✨

  • 8-directional movement
  • Weighted terrains (Grass = 2, Water = 5)
  • Real-time visualization
  • Animations + Stats (steps, path length, runtime)
